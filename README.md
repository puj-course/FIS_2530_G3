# 👗 Eco Moda – Plataforma de Moda Circular

**Tecnología con propósito social y ambiental**

---

## 📖 Descripción

Eco Moda es una plataforma web que facilita la compra, venta, intercambio y donación de ropa de segunda mano en buen estado. Su misión es reducir el impacto ambiental de la industria textil, fomentar el consumo responsable y apoyar a comunidades vulnerables mediante el acceso gratuito o económico a prendas de vestir.

Inspirada en la encíclica *Laudato Si’*, Eco Moda transforma el acto de consumir moda en una acción ética y sostenible.

---

## 🚀 Características Principales

- ♻️ Sistema de donaciones con comprobantes y trazabilidad.
- 🧍 Manejo de usuarios, perfiles y recompensas por participación.
- 💬 Chat en línea entre usuarios para facilitar intercambios.
- 💳 Integración con portales de pago.
- 🌐 Acceso multiplataforma desde cualquier dispositivo.

---

## 🧠 Arquitectura y Diseño

El sistema sigue una arquitectura cliente-servidor con enfoque modular. Se prioriza la escalabilidad, la accesibilidad y la interoperabilidad entre tecnologías.

### 🔷 Componentes

- **Frontend**: HTML, JavaScript  
- **Backend**: Node.js  
- **Base de datos**: MongoDB y MySQL  
- **Infraestructura**: Railway  
- **Control de versiones**: Git y GitHub

## 🔁 Metodología de Desarrollo

El proyecto se desarrolla bajo metodología ágil Scrum, con sprints semanales y planificación colaborativa. Se utilizan historias de usuario, sub-issues y milestones temáticos (donaciones, perfiles, recompensas, etc.).

### Herramientas de gestión

- Tablero Kanban: `Backlog`, `Ready`, `In Progress`, `Done`  
- Issues organizados por prioridad y complejidad  
- Documentación colaborativa en GitHub
